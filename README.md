### Installation

_Below is an example of how you can instruct your audience on installing and setting up your app. This template doesn't rely on any external dependencies or services._

1. Go to github [https://github.com](https://github.com)
2. Clone the repo
   ```sh
   git clone https://github.com/SorayuthJapanya/Booking-Meeting_Room-Management.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter scriptId in `.clasp.json`

   ```sh
   "scriptId": "Enter your scriptId",
   ```

5. Clone file in gas folder to google apps scrit

   ```sh
   clasp clone
   ```

6. Deploy google apps script for get GAS URL
7. Enter your Google App Script API, Cloudinary API and Cloudinary preset in `.env`
   ```env
    VITE_GAS_URL="Enter your gas url"
    VITE_CLOUDINARY_URL="Enter your Cloudinary url"
    VITE_CLOUDINARY_PRESET="Enter your preset"
   ```
